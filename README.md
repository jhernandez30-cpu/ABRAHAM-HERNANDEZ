## 💼 Proyectos destacados

### 🛡️ ITSA Segurity

Dashboard enfocado en visibilidad de amenazas en tiempo real.

**Problema:**  
Empresas sin monitoreo instantáneo de amenazas.

**Solución:**  
Dashboard con WebSockets para mostrar alertas en tiempo real.

**Stack:**  
React, Socket.IO, Node.js, MongoDB.

**Resultado:**  
Reducción del tiempo de respuesta ante incidentes.

---

### 📊 Dashboard Analytics

Dashboard interactivo para visualizar datos dispersos y facilitar la toma de decisiones.

**Stack:**  
Vue, D3.js, Express, PostgreSQL.

---

### 🤖 AI Chatbot

Chatbot orientado a atención al cliente 24/7 con capacidad de responder consultas comunes y escalar a un agente humano.

**Stack:**  
React, Python, Flask, Rasa, MongoDB.
